# E-commerce Sales Insights & Trends Using SQL

---

## Overview
This project analyzes e-commerce sales data using SQL to extract meaningful business insights. It focuses on revenue, order volume, product and category performance, city-wise demand, pricing segments, and monthly sales trends.

---

## Objectives
- Analyze overall sales and revenue trends  
- Identify top-performing products, brands, and categories  
- Compare performance across cities and platforms  
- Understand pricing-based revenue contribution  
- Track monthly sales trends and seasonality  

---

## Data
- E-commerce sales transaction dataset  
- Fields include order ID, product, category, brand, platform, city, price, quantity, total amount, rating, reviews, and order date  

---

## Analysis & Approach
- Stored and structured data in a SQL database  
- Performed exploratory analysis using aggregation and grouping  
- Used subqueries to compare performance against averages  
- Analyzed sales by product, category, brand, platform, and city  
- Conducted time-based analysis for monthly sales trends  

---

## Key Business Questions Answered

- Which products generate the highest revenue and sell the most units?  
- Which brands contribute the most to overall sales?  
- Which cities generate the highest revenue and sell more products?  
- How does revenue vary across pricing segments?  
- What are the monthly sales trends and seasonality patterns?
